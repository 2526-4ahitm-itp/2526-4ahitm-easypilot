== 2. Sprint Backlog und User Story


### User Story: Live-Telemetrie in 3D-Visualisierung

* Als Drohnenpilot möchte ich mittels einem Raspberr Pi eine Tonleiter bzw Melodie wie bei einer Taurus Lokomotive abspielen können, um zu Testen ob die Motoren individuell ansteuerbar sind.
### Definition of Done (Akzeptanzkriterien)

Die folgenden Kriterien müssen für die Abnahme der User Story erfüllt sein:

* Voreingespielte Sequenz von Tönen wird abgespielt.
* Jeder Motor kann individuell angesteuert werden.
* Die Tonhöhe und Dauer der Töne kann angepasst werden.
* Sequenzdauer ist inviduell anpassbar.
