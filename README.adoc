== ğŸ“˜ Project Description

*EasyPilot* is a modular DIY drone/autopilot platform built around a Raspberry Pi.
It provides real-time telemetry, camera streaming, ESC/motor control, optional YOLOv8 object detection, and a safe experimentation environment for autonomous flight.

Main features:

* Raspberry Pi as the main compute unit
* Live video streaming (RTSP/MJPEG)
* Telemetry via WiFi or LoRa
* ESC/motor output control
* IMU-based stabilization (MPU6050/9250)
* Optional YOLO object detection pipeline
* Expandable ground-station interface
